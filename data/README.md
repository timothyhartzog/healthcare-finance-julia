# Data

Synthetic datasets for teaching and testing healthcare finance models.

## Files

| File | Description | Rows |
|---|---|---|
| `synthetic/inpatient_discharges.csv` | MS-DRG hospital discharges with charges, payments, payer mix | 20 |
| `synthetic/medicare_advantage_members.csv` | MA member-level HCC risk scores and annual claims | 20 |
| `synthetic/budget_actual.csv` | Department-level budget vs. actual for 4 months | 16 |
| `synthetic/hospital_financials.csv` | System-level income statement and balance sheet items | 5 |

All data is **fully synthetic** — no patient or provider identifiers are real.
Values are calibrated to be plausible for teaching purposes.


This directory contains schemas, synthetic datasets, and ETL workflows for healthcare finance analysis.
